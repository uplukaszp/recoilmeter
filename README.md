# recoilmeter
Recoilmeter based on mpu6050 and arduino


# connections

## mpu6050

MPU6050	-	ESP12E
VCC		-	3V
GND		-	GND
SDA		-	D2 (SDA)
SCL		- 	D1 (SCL)


## keyboard
keyboard	-	ESP12E
1			-	GND
2			-	D4
3			-	D3

## sd card reader
reader  - ESP12E
GND     - GND
3V      - 3V
CS      - D8 
MOSI    - D7
SCK     - D5
MISO    - D6
